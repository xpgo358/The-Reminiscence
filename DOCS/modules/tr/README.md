# TR - Indice del Modulo

## Alcance
El namespace tr concentra los sistemas sociales y de coordinacion de jugadores del servidor:
- Estado AFK por jugador y por dimension.
- Nivel de prank y su representacion visual.
- Estados de streaming y recording.
- Triggers de informacion (help, rules, version).
- Selector de color hexadecimal para locator bar.
- Operaciones administrativas (scoreboards, teams y arranque de challenge announcements).
