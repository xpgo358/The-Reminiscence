# TR: challenges

El subsistema de challenges sirve para anunciar contenido especial con una secuencia de carga dramática.

## Entrada principal

- `tr:admin/challenges/new`

## Qué hace

La función prepara una pequeña presentación en varios pasos con:

- Mensajes de carga.
- Sonido de impacto.
- Título principal destacado.
- Subtítulo explicativo.
- Action bar final.

## Intención de diseño

El objetivo no es solo informar de que existe un nuevo challenge. También busca darle peso visual y sonoro para que el anuncio se sienta como un evento dentro del servidor.

## Cuándo usarlo

Se usa cuando el operador quiere introducir un reto nuevo, una dinámica temporal o un anuncio importante vinculado al juego.

## Consideración técnica

La secuencia está fragmentada en varios pasos para que el ritmo se perciba en pantalla y no como un mensaje plano único.
